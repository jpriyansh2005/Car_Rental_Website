A fully responsive Car Rental Website built using the MERN (MongoDB, Express, React, Node.js) stack.
The website allows users to browse luxury cars, check availability, and rent cars with a simple and interactive UI.

Frontend:

React.js

Tailwind CSS

React Router

Axios

Backend:

Node.js

Express.js

JWT Authentication

Database:

MongoDB (Mongoose ORM)
